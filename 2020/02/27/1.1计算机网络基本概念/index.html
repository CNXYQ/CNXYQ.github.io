1.1计算机网络基本概念

计算机网络基本概念

定义：计算机网络是互连的、自治的计算机集合。

  • 自治-无主从关系

  • 互连-互联互通

    通过交换网络互连主机

计算机网络：

​ 主机 <- - - - -> 交换网络（由交换节点组成（路由器或交换机））<- - - - >主机

什么是Internet?

A.从组成细节角度

全球最大的互联网络

  • ISP网络互连的“网络之网络”
  • 数以百万计的互连的计算设备集合
    • 主机（hosts）= 端系统（end system）
    • 运行各种网络应用
  • 通信链路
    • 光线，铜缆，无线电，卫星。。。
  • 分组交换：转发分组（数据包）
    • 路由器（routers）和交换机（switches）

B.从服务角度

  • 为网络应用提供通信服务的通信基础设施:
    • Web,VoIP,email,网络游戏，电子商务，社交网络。。。
  • 为网络应用提供应用编程接口（API）:
    • 支持应用程序“连接”Internet，发送/接收数据
    • 提供类似与邮政系统的数据传输服务

什么是网络协议？(协议是计算机网络有序运行的重要保证)

网络协议（协议）是进行网络中的数据交换而建立的规则，标准或约定

协议规定了通信实体之间所交换的消息的格式，意义，顺序以及针对信息或发生的时间所采取的“动作”

  • 硬件（主机，路由器，通信链路等）是计算机网络的基础
  • 计算机网络中的数据交换遵守事先约定好的规则

（任何通信或信息交换过程都需要规则）

网络通信:

  • 通信主体是“机器”而不是人

  • 交换“电子化”或“数字化”消息

  • 计算机网络的所有通信过程都遵守某种/些规则-协议

协议的三要素：

1. **语法**（Syntax）
- 数据与控制信息的结构或格式
- 信号电平
2.  **语义**（Semantics）
- 需要发出何种控制信息
- 完成何种动作以及做出何种响应
- 差错控制
3. **时序**（Timing）
- 事件顺序
- 速度匹配

协议是计算机网络的重要内容

  1. 协议规范了网络中所有信息发送和接收过程

    • e.g.，TCP，IP，HTTP，Skype，802.11

  2. 学习网络的重要内容之一

  3. 网络创新的表现形式之一

  4. Internet协议标准

    • RFC：Request for Comments
    • IETF： 互联网工程任务组(Internet Engineering Task Force)
