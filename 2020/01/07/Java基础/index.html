      /**

      用&去解决以此来验证”&“的作用

      */
      public class 验证按位与{

      public static void main(String args[]){
    System.out.println(10 != 10 & 10 / 0 == 0);
}

      程序的执行结果:
      造成以上问题的根本原因就是与操作要把所有的条件都进行判断，所以在计算10 / 0时就出现了错误 ,那么修改为短路与呢。见下面的代码

      /**

      用&&去解决以此来验证”&&“的作用

      */
      public class 验证双位与{

      public static void main(String args[]){
    System.out.println(10 != 10 && 10 / 0 == 0);
}

      }
      上面的程序运行后，不再出现以上的错误提示，因为第一个条件不满足，所以后面的条件就不再判断了，这就是短路与的作用。

      /**

      用|去解决以此来验证”|“的作用

      */
      public class 验证按位或{

      public static void main(String args[]){
    System.out.println(10 == 10 | 10 / 0 == 0);
}

      }
      造成以上问题的根本原因就是与操作要把所有的条件都进行判断，所以在计算10 / 0时就出现了错误 ,那么修改为短路或呢。见下面的代码

      /**

      用||去解决以此来验证”||“的作用

      */
      public class 验证双位或{

      public static void main(String args[]){
     System.out.println(10 == 10 || 10 / 0 == 0);
 }

      }

      从上面的运行结果可以发现，程序不再出现错误，就是因为使用了 “短路或” 如果第一个条件为true，则后面的条件将不再判断。

      规律如下:

      (1) 对于短路与(&&)也就是双位与而言。同真则真，一假则假。存在短路特性(前面表达式为假，后面就不需要判断了)

比如说：10 != 10 && 10 / 0 == 0

(2) 对于非短路与(&)也就是按位与而言。同真则真，一假则假。不存在短路特性(不管前面的表达式是否成立，后面的都要执行)

比如说：10 != 10 & 10 / 0 == 0
        (3) 对于短路或(||)也就是双位或而言。同假则假，一真则真。存在短路特性(前面表达式为真，后面就不需要判断了)

        比如说：10 == 10 || 10 / 0 == 0


        （4）对于非短路或(|)也就是按位或而言。同假则假，一真为真。不存在短路特性(不管前面的表达式是否成立，后面的都要执行)

        比如说：10 != 10 | 10 / 0 == 0


        重点掌握短路与和短路或

        也就是说对于短路与和短路或，第一个条件就是关键性条件，所以在Java设置了短路与和短路或，可以提高程序的运算效率。

/**
位运算符：按位与(&)的案例
*/
public class 位运算符按位与{
public static void main(String args[]){
int a = 128;
int b = 129;

              System.out.println("a&b的结果为:"+(a&b));
       }
   }
   a&b的结果为：128
   下面来分析这个程序：
       a的值是128，转换为二进制10000000
       b的值是129，转换为二进制10000001
       两个二进制数据对应起来，根据按位与运算的规律可以知道，结果为1000000
       10000000
       10000001
       ----------------
       10000000

       只有对应的位都为1，结果才是1，结果为10000000

       但结果为什么是128呢？而不是10000000呢？因为10000000这个结果是针对a&b在计算机内部存储的计算结果。而最终显示出来的结果是以计算得到的二进制数10000000对应的十进制进行显示的。所以说得到的结果128。

       这个里面涉及到了一个进制的转换  二进制转换为十进制
