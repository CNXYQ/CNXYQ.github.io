1.4 算数赋值运算符号

  1. 作用：先运算，再赋值

  2. 代码演示

    1. /**
  *  声明两个整型变量i和j，分别赋值10和2，针对变量i和j进行加、减、乘、除、余的运算，但是将每一次运算得到的结果再赋值给变量i，然后用新i的值再和j进行其它运算，最终打印出i的值 
  */
public class 算数赋值运算符号1{
    public static void main(String args[]){
         int i = 10;
         int j = 2;

         i = i + j;
        i = i - j;
        i = i * j;
        i = i / j;
        i = i % j;

        System.out.println("i="+j);
     }
}

/**
    利用算数赋值运算符号来解决上述问题
  */
      public class 算数赋值运算符号2{
    public static void main(String args[]){
         int i = 10;
         int j = 2;

         i += j;//i = i + j
         i -= j;
         i *= j;
         i /= j;
         i %= j;

           System.out.println("i="+j);
     }
}

    2. /**
    用&&去解决以此来验证”&&“的作用
  */
public class 验证双位与{
    public static void main(String args[]){
        System.out.println(10 != 10 && 10 / 0 == 0);
    }
}
上面的程序运行后，不再出现以上的错误提示，因为第一个条件不满足，所以后面的条件就不再判断了，这就是短路与的作用。
<!--￼12-->
    3. /**
    用||去解决以此来验证”||“的作用
  */
public class 验证双位或{
    public static void main(String args[]){
         System.out.println(10 == 10 || 10 / 0 == 0);
     }
}

从上面的运行结果可以发现，程序不再出现错误，就是因为使用了 “短路或” 如果第一个条件为true，则后面的条件将不再判断。

  1. 按位或(|)

    1. /**
    位运算符：按位(|)的案例
  */
public class 位运算符按位或{
    public static void main(String args[]){
        int a = 128;
        int b = 129;
        System.out.println("i|j的结果为:"+(a|b));
    }
}

a|b的结果为：129
下面来分析这个程序：
    a的值是128，转换为二进制10000000
    b的值是129，转换为二进制10000001
    两个二进制数据对应起来，根据按位与运算的规律可以知道，结果为10000001
    10000000
    10000001
----------------------
    10000001

    只要对应的位有一个1，结果就是1，如果都为1，结果也为1，所以说结果为10000001

    但结果为什么是129呢？而不是10000001呢？因为10000001这个结果是针对a|b在计算时，在计算机内部存储的计算结果。而最终显示出来的结果是以计算得到的二进制数10000001对应的十进制进行显示的。所以说得到的结果129。而不是计算的到的二进制数据10000000

    这个里面涉及到了另一个进制的转换  二进制转换为十进制

        右移两位之后变成 00001111
        转换为十进制显示为：15

1.9 括号运算符号

  1. 代码演示

    1. /**
    括号运算符号案例
  */
public class 括号运算符号{
    public static void main(String args[]){

        int i = 3 + 5 + 4 * 6 -7;
        int j = ((3 + 5 + 4) * (6 - 7));

        System.out.println("3 + 5 + 4 *6 -7="+i);
        System.out.println("((3 + 5 + 4) * (6 - 7))="+j);

    }
}
程序的运算结果：
3 + 5 + 4 *6 -7 = 25
((3 + 5 + 4)* (6 - 7)) = -12
