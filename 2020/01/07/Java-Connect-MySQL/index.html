1.下载驱动包

​ 根据自己的MySQL版本下载对应的驱动包，连接:百度网盘

2.在java中导入驱动包并添加至构建路径

  1. 新建项目，包，类（这个我就不说了）

  3. 点击粘贴好的驱动包右键选择添加至构建路径
